# VK Chat Game Bot

Бот для бесед ВКонтакте, который:
- выбирает "красавчика дня"
- выбирает "пидора дня"
- ведёт статистику побед
- показывает предсказания
- делает анимационные отсчёты

---

##  Установка

git clone https://github.com/yourname/vk-bot.git
cd vk-bot
pip install -r requirements.txt

##  Настройка

Создай переменные окружения:

export VK_TOKEN=your_token_here

Или используй .env.

##  Запуск
python bot.py

##  Команды бота
/красавчик — выбрать красавчика дня

/пидор — выбрать "пидора дня"

/предсказание — случайное предсказание

/статистика — рейтинг участников

/ — список команд

##  Особенности
Long Poll API VK
cooldown 24 часа на команды
сохранение статистики в JSON
случайный выбор участников
анимационные сообщения

##  Важно
Бот должен быть администратором беседы,
иначе он не сможет получить список участников.
